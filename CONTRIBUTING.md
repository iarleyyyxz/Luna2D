### 1. Fork the Repository
Click the **Fork** button at the top right of the repository page and clone your fork locally:

```bash
git clone https://github.com/your-username/luna-game-engine.git
cd luna-game-engine
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

Branch naming convention:
- feature/
- fix/
- docs/
- refactor/

### Code Standards

- Language: C#

- PascalCase for public members

- camelCase for private fields

- Keep methods focused and readable

- Avoid unnecessary allocations

- Prefer data-oriented design when possible

### Commit Messages Guideline
- "feat: add animation blend trees"
- "fix: resolve sprite flickering issue"
- "refactor: improve renderer batching"

### Testing
Before creating a pull request:

- Build the project successfully

- Run the engine and test changes

- Ensure no console errors occur

### Pull Request Process

- Push your branch

- Open a PR

##### Describe clearly:

- What changed

- Why the change was needed

- Screenshots or videos if applicable
